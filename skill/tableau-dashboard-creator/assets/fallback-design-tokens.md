# Fallback Design Tokens Pointer

The canonical fallback token reference lives at `references/tableau-design-tokens.md`.

Use that file as the single source of truth for:
- default typography
- default colors
- default dashboard sizing
- default container hierarchy
- default Tableau fidelity constraints

When Step 0 uses any fallback values, the generated `design-tokens.md` should disclose each one under `## Fallback Decisions`.
